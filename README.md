# Master Thesis
This is my Master Thesis at ETHZ

# Primary Objective
To design and implement a foundational (possibly monadic) framework in Lean 4 that strictly formalizes the analysis of randomized algorithms. The ultimate goal is to create a seamless translation pipeline where the Lean 4 codebase mirrors as possible the structure and readability of a standard LaTeX mathematical proof, specifically allowing for rapid, multi-line verification of running times and probability bounds.

## Toolchain
Built with `leanprover/lean4:stable`.
